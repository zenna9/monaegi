# Monaegi (모내기)

GitHub 커밋을 활용한 목표 달성 및 잔디 시각화 프로젝트입니다.

## 주요 기능
- **GitHub 로그인**: GitHub 계정으로 로그인하고 전용 빈 레포지토리(저장소)를 생성합니다.
- **목표 설정**: 특정 기간에 대한 큰 목표를 설정하고, 이를 쪼개어 상세 세부 목표를 설계합니다.
- **잔디 심기 (커밋)**: 세부 목표 달성 및 공부 내역을 기록하면, 연동된 레포지토리에 커밋되어 GitHub 잔디(Contribution graph)에 반영됩니다.

## 아키텍처 및 기술 스택
- **프론트엔드**: Next.js (App Router), React, TypeScript, Tailwind CSS
- **백엔드 / 데이터베이스**: 없음 (No-BE, No-DB)
- **데이터 저장 방식**: 모든 데이터(진행률, 목표 등)는 사용자의 GitHub 레포지토리 내에 JSON 파일 형태로 저장됩니다. 프론트엔드에서 GitHub API (Octokit 등)를 호출하여 직접 CRUD를 수행합니다.
